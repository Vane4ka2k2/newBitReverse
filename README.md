# newBitReverse

Интерактивное десктопное приложение на Qt 6 для захвата видеопотока с веб-камеры в реальном времени и применения различных фильтров обработки изображений. Вычисления вынесены в отдельный поток с использованием архитектуры воркеров для обеспечения высокой отзывчивости графического интерфейса (UI).

## 🎥 Особенности
- **Захват видеопотока в реальном времени**: Работа с камерой с использованием мультимедийного стека Qt 6 (`QCamera`, `QVideoSink`, `QMediaCaptureSession`).
- **Асинхронная обработка (Multithreading)**: Кадры передаются в фоновый поток `QThread` через систему сигналов и слотов, что предотвращает зависание интерфейса при выполнении тяжелых математических операций.
- **Интегрированные фильтры**:
  - **Grayscale**: Преобразование в оттенки серого.
  - **Sobel Operator**: Выделение границ объектов (оператор Собеля).
  - **Pad to Power of Two**: Дополнение разрешения кадра до ближайшей степени двойки (требуется для корректной работы алгоритма битового реверса).
  - **Pixelate**: Эффект пикселизации (с размером блока 8px).
  - **Threshold (Бинаризация)**: Пороговое преобразование изображения в двухцветный черно-белый формат.
  - **Bit Reverse (Битовая перестановка)**: Перестановка координат пикселей на основе побитового реверсирования индексов.

## 🛠️ Стек технологий
- **Язык**: C++ (стандарт C++17)
- **Фреймворк**: Qt 6.x (модули `Widgets`, `Multimedia`)
- **Система сборки**: qmake (файл проекта `test_stb_webcamera.pro`) / CMake

## 📂 Структура проекта
- `main.cpp` — входная точка приложения, инициализация Qt-приложения.
- `MainWindow.h` / `MainWindow.cpp` — класс главного окна, отображение интерфейса, управление горячими клавишами и инициализация камеры.
- `ProcessorWorker.h` — класс воркера для фоновой потоковой обработки кадров.
- `h/Image.h` / `cpp/Image.cpp` — кастомный класс `Image` для низкоуровневых операций с пикселями (конвертация, фильтры, реверс координат).
- `h/stb_image.h` / `h/stb_image_write.h` — сторонние библиотеки stb для загрузки/сохранения изображений.

## ⌨️ Управление
Приложение поддерживает переключение фильтров "на лету" с помощью горячих клавиш:
- `1` — Вкл/Выкл фильтр **Bit Reverse** (побитовая перестановка координат).
- `2` — Вкл/Выкл фильтр **Threshold** (бинаризация с порогом 127).
- `3` — Вкл/Выкл фильтр **Pixelate** (пикселизация).
- `4` — Вкл/Выкл фильтр **Sobel** (выделение границ Собеля).
- `5` — Вкл/Выкл фильтр **Grayscale** (оттенки серого).

## 📦 Сборка и запуск

Для сборки проекта вам понадобится установленный Qt 6 (с модулем Multimedia) и компилятор (например, MinGW 64-bit или MSVC).

### Через Qt Creator:
1. Откройте файл `test_stb_webcamera.pro` в среде Qt Creator.
2. Настройте комплект сборки (Kit) с версией Qt 6.
3. Нажмите кнопку **Run** (или `Ctrl + R`) для автоматической компиляции и запуска приложения.

### Через командную строку (qmake):
```bash
# Создайте директорию сборки
mkdir build
cd build

# Вызовите qmake для генерации Makefile
qmake ../test_stb_webcamera.pro

# Соберите проект
make # или mingw32-make / nmake в зависимости от компилятора
```
